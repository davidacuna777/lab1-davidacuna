## 1. Contexto del Proyecto

Este repositorio contiene la investigacion, diseno y desarrollo de un portafolio web personal orientado a la postulacion para un puesto ficticio de **Junior Web Developer**.

El sitio fue desarrollado con:

- HTML5
- CSS3
- JavaScript (uso limitado y justificado)

No se utilizaron plantillas ni frameworks (Bootstrap, Tailwind, React, etc.) para cumplir con las restricciones del enunciado.
